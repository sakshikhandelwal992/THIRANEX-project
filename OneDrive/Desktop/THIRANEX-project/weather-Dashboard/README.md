# 🌦️ Weather Dashboard

A responsive Weather Dashboard built using **HTML, CSS, and JavaScript** that fetches real-time weather information from the OpenWeather REST API. Users can search for any city and instantly view live weather details such as temperature, humidity, wind speed, and weather condition.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Display real-time temperature
- 💧 Show humidity percentage
- 🌬️ Display wind speed
- ☁️ Show current weather condition
- ⚡ Fetch live weather data using Fetch API
- 🔄 Uses Async/Await for asynchronous requests
- 📦 Parses JSON response from REST API
- ❌ Handles invalid city names
- ⚠️ Handles network and runtime errors using try...catch
- 📱 Responsive user interface

## ⚙️ How It Works

1. User enters a city name.
2. JavaScript captures the input.
3. A request is sent to the OpenWeather REST API.
4. The API returns weather data in JSON format.
5. The JSON data is parsed.
6. Weather information is dynamically displayed on the webpage.
7. Errors such as invalid city names or network failures are handled gracefully.
